Country Profiles

Access profiles of countries, dependent territories, and special areas of geographical interest. Profiles are accessed one-by-one by the country’s name, two or three-letter abbreviation, a numerical identifier, or top-level domain.

The information is sourced from The World Factbook, published by the CIA in the public domain. To explore The World Factbook in its original form, please use CIA’s website. For more information, see this Wikipedia article.

Base URL

http://api.sollinda.com/v1/country-profiles

opendatajson/factbook.json is used to generate this API. Many thanks to the contributors of factbook.json for making this possible.

Single Country Profile

# Using type "name"
TYPE=name
ID=taiwan

# Using type "iso-alpha-2"
TYPE=iso-alpha-2
ID=tw

# Using type "iso-alpha-3"
TYPE=iso-alpha-3
ID=twn

# Using type "tld"
TYPE=tld
ID=.tw

HTTP Request Format

GET http://api.sollinda.com/v1/country-profiles/<type>/<id>

Request Parameters

Parameter
type The Country Identifier Type used to identify the country.
id The ID of a country for the given type parameter. The ID should be lowercased even if the standard in type uses a different convention.
Refer to the list of Country Identifier Types for all supported values for the type parameter.

Country Identifier Types

Country profiles are identified by using the country name or a regognized standard such as the two-letter country code used in ISO 3611-1 alpha-2.

For more information about the standards, refer to the links in the description fields. Or, for a list of all countries with identifiers, use the List of Country Profiles endpoint.

Type Parameter
name The lowercased and underscore-separated name of a country.
E.g. taiwan for Taiwan or united_states for USA.
iso-alpha-2 ISO 3611-1 alpha-2 codes are two-letter country codes as defined in ISO 3166-1. They are most prominently used for Internet’s country-specific top-level domains, but also by banks (IBAN and SWIFT), United Nations, and others.
E.g. tw for Taiwan, us for USA, au for Australia.
iso-alpha-3 ISO 3611-1 alpha-3 codes are similar to alpha-2 but uses three letters instead of two.
E.g. chn for China and sgp for Singapore.
iso-numeric ISO 3611-1 numeric uses three-digit country codes.
E.g. 158 for Taiwan or 250 for France.
gec Geopolitical Entities and Codes (GEC) is used by The World Factbook as primary country identifier. Note that this standard differens from the more widely used ISO-3611-1 alpha-2.
E.g. tw for Taiwan, au for Austria.
stanag STANAG 1059 is a three-letter identifier established and maintained by NATO for the purpose of providing a common set of geo-spatial identifiers for countries and territories.
E.g. twn for Taiwan, tha for Thailand.
tld Top-level Domains, e.g. .se for Sweden or .tw for Taiwan.
